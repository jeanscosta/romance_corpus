## Overview

This repository contains the full analysis code and supplementary materials for a corpus-based study of boundary-crossing motion event encoding in the written English of Romance-language learners (French, Italian, Portuguese, Spanish) across five CEFR proficiency levels (A1–C1).

The study investigates two competing strategies for encoding boundary-crossing events in English:

- **Satellite-framed constructions** (target in English): *manner verb + into* (e.g., *walk into the room*) and *manner verb + out of* (e.g., *run out of the building*)
- **Verb-framed constructions** (L1-transferred from Romance languages): *enter* and *exit* with explicit spatial Ground nouns (e.g., *entered the building*)

The central theoretical prediction, derived from Talmy's (1985, 2000) motion event typology and Slobin's (1996) thinking-for-speaking framework, is that Romance-language learners will underuse the satellite-framed strategy at lower proficiency levels, with rates increasing as proficiency advances.

A key methodological contribution is the application of a **two-step semantic filtering procedure** to ensure that only instances with physically bounded spatial Ground nouns are counted, addressing a systematic source of noise in corpus-based motion event research.

## Data

### Corpus
The analysis uses a learner corpus of written English compiled from a massive open online course (MOOC). The corpus comprises 20 sub-corpora (4 L1 groups x 5 CEFR levels), totalling 13,014,386 words.

| L1 | A1 | A2 | B1 | B2 | C1 | Total |
|---|---|---|---|---|---|---|
| French | 283,933 | 407,156 | 330,678 | 156,287 | 53,745 | 1,231,799 |
| Italian | 219,170 | 443,686 | 386,307 | 191,470 | 80,823 | 1,321,456 |
| Portuguese | 2,678,593 | 2,890,234 | 1,854,464 | 720,301 | 259,168 | 8,402,760 |
| Spanish | 705,924 | 646,157 | 535,951 | 136,370 | 33,969 | 2,058,371 |

### POS Tagging Format
Corpus files must be POS-tagged using the **Penn Treebank tagset**, with each token in `word_TAG` format, space-separated. Example:

```
She_PRP walked_VBD slowly_RB into_IN the_DT room_NN ._.
The_DT referee_NN threw_VBD the_DT balloon_NN into_IN play_NN ._.
```

Verb tokens are identified using tags: `VB`, `VBZ`, `VBP`, `VBD`, `VBG`, `VBN`.

## Usage

### 1. Configure file paths

Edit the `FILES` dictionary in Section 1 of `boundary_crossing_analysis.py`:

```python
FILES = {
    ('french', 'A1'): 'path/to/ST_french_A1_all.txt',
    ('french', 'A2'): 'path/to/ST_french_A2_all.txt',
    # ... etc.
}
```

### 2. Run

```bash
python boundary_crossing_analysis.py
```

Five steps run in sequence with console output. Typical runtime: 2-10 minutes depending on corpus size.

### 3. Review concordances

Inspect the three concordance files to verify retrieved instances. Document decisions in the manual review files.

## Semantic Filtering

The filtering procedure addresses a systematic problem in corpus-based motion event research: syntactic search retrieves non-literal uses of the target constructions.

### Sources of noise identified in this corpus

| Source | Construction | n excluded | Example |
|---|---|---|---|
| Sports idiom | MV+into | 44 | *throw into play* |
| Body-part grounds | MV+into | 122 | *push into the stomach* (narrative task) |
| Abstract grounds | MV+into / PV | ~50 | *enter the job market*, *run out of time* |
| Misspelled *exciting* | PV (*exiting*) | 6 | *very exiting Hi!* |
| Intransitive PV | PV | 788 | *she entered* (no overt Ground noun) |

### Extending the lexicon

To apply the script to a new corpus, run once to collect unknown Ground nouns, review concordances, add verified spatial nouns to `SPATIAL_GROUNDS` in Section 4, then re-run for final counts.

## Statistical Methods

| Test | Purpose | Rationale |
|---|---|---|
| **Jonckheere-Terpstra** | Developmental trend (H1) | Tests monotonic increase specifically; designed for ordered alternatives; no normality assumption |
| **Spearman correlation** | Level-frequency association | Robust to outliers; no linearity assumption; pooled and per-language |
| **Kruskal-Wallis** | L1 group differences (H2) | Non-parametric ANOVA equivalent |
| **Partial Spearman** | MV vs. PV relationship | Controls for shared proficiency effect via rank residualization |

Negative binomial regression was explored but produced unstable estimates given the small post-filtering counts (MV+into n=171, PV n=201 across 20 cells).

## Adapting to Other Languages or Constructions

- **Different L1 groups**: edit `LANGUAGES` and `FILES`
- **Different proficiency scales**: edit `LEVELS` and `LEVEL_MAP`
- **Additional constructions**: extend `search_file()` following the existing pattern
- **Different verb lists**: edit `SELF_MOTION_VERBS`, `CAUSED_MOTION_VERBS`, and `LEMMA_MAP`
- **Different Ground lexicons**: edit `SPATIAL_GROUNDS` and `EXCLUDE_NOUNS`
- **Different POS tagger**: update `VERB_TAGS`, `DET_TAGS`, and token parsing in `search_file()`
